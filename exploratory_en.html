Initial exploration of RBS data and understanding of network characteristics

Data Overview

The exploratory analysis was conducted with radio base station (RBS) data from different operators. The data includes information about geographic location, technical parameters, signal strength, and coverage quality.

Signal Quality Analysis

Signal quality was analyzed as a function of RBS distance, station density, and terrain characteristics. The results indicate important correlations between these factors and coverage quality.

Correlations and Patterns Found

Several important correlations were identified between network metrics, geographic features, and signal quality. These correlations provide valuable insights for predictive modeling and network optimization.

Figure 8: Correlation matrix between the main variables analyzed

Key Findings

  • Strong negative correlation (-0.78) between distance to nearest RBS and signal quality
  • Areas with higher RBS density show better coverage but also higher interference
  • Topographic features have significant impact on signal propagation
  • Communities detected in the network correspond to specific geographic areas
  • Critical nodes identified whose failure could significantly compromise coverage

Next Steps

Based on the exploratory analysis, the next step is the implementation of predictive models based on Graph Neural Networks (GNNs) to predict coverage quality and identify optimal locations for the installation of new RBSs.

# Excerpt from GNN data preparation
import networkx as nx
import torch
from torch_geometric.data import Data

def prepare_graph_data(rbs_data, connections):
    G = nx.Graph()
    
    # Add nodes (RBSs)
    for idx, rbs in rbs_data.iterrows():
        G.add_node(idx, 
                  lat=rbs['latitude'], 
                  lon=rbs['longitude'],
                  power=rbs['power_dbm'],
                  height=rbs['antenna_height'])
    
    # Add edges (connections between RBSs)
    for source, target, weight in connections:
        G.add_edge(source, target, weight=weight)
        
    return G
